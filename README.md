# 🎯 Chiba Portfolio

A modern personal portfolio website built with React and Vite, showcasing my projects, skills, and professional experience in a sleek and engaging manner.

## ✨ Key Features

- 🎨 **Modern Design** — Elegant user interface with smooth animations and transitions
- ⚡ **High Performance** — Built on Vite for lightning-fast loading times
- 📱 **Fully Responsive** — Works perfectly on all devices and screen sizes
- 🎭 **Dark Theme** — Professional dark design with Tailwind CSS
- 🔄 **Smooth Animations** — Beautiful motion effects with the Motion library
- 📝 **Comprehensive Sections** — Home, About, Skills, Projects, and Contact

## 🏗️ Project Structure

```
src/
├── sections/          # Main page sections
│   ├── Home/         # Landing page with hero section
│   ├── About/        # About me section
│   ├── Skills/       # Technical skills showcase
│   ├── Projects/     # Portfolio of previous projects
│   ├── Contact/      # Contact form section
│   └── Footer/       # Website footer
├── components/
│   ├── ui/           # Reusable UI components
│   └── custom/       # Custom components
└── style/            # Styling and CSS files
```

## 🛠️ Tech Stack

- **React 19** — JavaScript library for building user interfaces
- **Vite** — Next-generation frontend build tool
- **Tailwind CSS** — Utility-first CSS framework for styling
- **Motion** — Animation library for smooth transitions
- **Lucide React** — Beautiful and consistent icon library

## 🚀 Getting Started

### Prerequisites
- Node.js 16 or higher
- npm or yarn package manager

### Installation & Setup

```bash
# Install dependencies
npm install

# Start the development server with HMR
npm run dev

# Build the project for production
npm run build

# Preview the production build
npm run preview
```

## 📋 Available Commands

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server with Hot Module Reload |
| `npm run build` | Build optimized production bundle |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint to check code quality |
